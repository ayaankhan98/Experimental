Setting Up CCMS (Practice Implementation)

So for All this i am assuming you all have python 3.6 or above and Git installed on your system.

First step is to create a virtual Environment.

Windows -

Open Command Prompt and Type

virtualenv venv

this will install some necessary files like Setuptools etc, and finally the virtual environment is created, now if you type

dir

to list all the directories you will notice that a new directory with name venv is created.Well, let's move into the venv directory by

cd venv

the next step is to activate this virtual environment, to activate this venv type

venv\Scripts\activate

after activating the venv let's clone the github repository of ccms, to clone the repository type the following git command

git clone https://github.com/ayaankhan98/ccms.git

after successful cloning install the necessary dependencies of ccms by running a pip command

pip install -r requirements.txt

after the installation of necessary dependencies, we have to tell the system that which application we want to run on the flask server, so to do this type the command

set FLASK_APP=application.py

now set the flask server into development mode (by default the flask server works in production mode, in production mode whenever you make anychange in anyfile of the project you have to restart the server every time you make a change, so to avoid this irritation switch to development mode)using the command

set FLASK_ENV=development

now finally it's the time to run the flask app, to run type the command

flask run

After this you will see something like this

* Serving Flask app "application.py" (lazy loading)
* Environment: development
* Debug mode: on
* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
